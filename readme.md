# Elit d'Elitas

> 마인크래프트 서버 『무릉도원』을 모티브로 한, 돈버는 게임

## 칭호

관리자 칭호
* 건설자
* 관리청

특수 칭호
* 농부

성장 가능 칭호
* 청장

## 명령어

* [ ] `도움말`
  * 사용할 수 있는 명령어의 목록을 보여준다.


* [x] `추천인 <디스코드 아이디>`
  * 자신의 추천인을 등록합니다.


* [x] `돈`
  * 현재 가지고 있는 돈, 접속코인을 확인합니다.
* [x] `돈 설정 <사람> <양>`*
  * `<사람>`의 소지금을 `<양>`으로 설정합니다.
* [x] `돈 주기 <사람> <양>`
  * 현재 가지고 있는 돈의 일부를 `<사람>`에게 줍니다.


* [ ] `인벤토리`
  * 인벤토리에 가지고 있는 아이템을 확인합니다.
* [ ] `작물정보 <작물>`
  * 작물 정보를 확인합니다.
  * 작물이 다 익기까지 걸리는 시간, 썩는 시간, 가격 등


* [ ] `밭`
  * 자신의 밭을 멘션하여 이동할 수 있도록 합니다.
* [ ] `밭 정보`
  * 자신의 밭 상태를 확인합니다.
  * 밭에 뭐가 심어져있는지 확인
  * 밭 최대 크기와 현재 밭 크기 등
  * 저장 가능한 아이템 양 등
* [ ] `밭 심기 <작물>`
  * 밭에 `<작물>`을 심습니다.
  * 어디에 심는지, 얼마나 심는지 물어봐야 함
  * `돈나무`를 심어서 밭에 돈을 보관할 수 있다.
* [ ] `밭 수확 <주소>`
  * `<주소>` 위치에 있는 작물을 수확합니다.
  * 주문서 등을 사용해서 속도를 빠르게 하거나 산화도를 낮춤
* [ ] `밭 확장`
  * 자신의 밭을 확장합니다.
  * 흙과 물이 80:1 비율로 필요합니다. 밭을 80칸씩 확장합니다.


* [ ] `상점`
  * 상점을 멘션하여 이동할 수 있도록 합니다.

## 아이템 타입

모든 아이템의 설명의 첫 줄은 아이템에 대한 데이터로 이루어져 있다.

* [ ] `1` 수표
  * 돈을 아이템화한 것
  * 사용하면 가치만큼의 돈을 얻을 수 있다.
  * 데이터: 수표 가치(정수) `/\d+/`
* [ ] `2` 작물
  * 밭에서 얻으면 인벤토리에 얻어지는 것
* [ ] `3` 주문서
  * 여러가지 활동들을 쉽게 해주는 것
  * 데이터: *구상 중*
